Rate My Whatever adds the option to rate your posts.

== Description ==

Rate My Whatever enables you to add a rating system to your wordpress posts.
Simply download, install and add [ratemywe] to your posts.

The plugin will remake [ratemywe] to a div with rate buttons and percentage of votes.

Your users can only vote 1 time per IP and post.

Suggestions or questions? Email me at; henric-johansson [at] hotmail [dot] com or use the contact form at:
http://www.keklabprogramming.com

== Installation ==

1. Upload `rate-my-whatever.zip` to the `/wp-content/plugins/` directory
2. Unzip the plugin.
3. Activate the plugin through the 'Plugins' menu in WordPress
